# App de Consulta de Locomotivas

Este aplicativo foi desenvolvido para otimizar o monitoramento e a gestão de locomotivas em campo, permitindo consultas rápidas, análise de falhas críticas e compartilhamento ágil de informações.

## 🚀 Funcionalidades
- **Busca Rápida:** Consulta de ativos e notas por busca textual.
- **Raio-X Crítico:** Análise automática de termos (ex: "jumper", "vandalismo", "fuelink") para identificar problemas prioritários.
- **Integração Operacional:** Compartilhamento direto das ocorrências via WhatsApp.
- **Gestão de Dados:** Interface administrativa para atualização rápida da planilha operacional.

## 🛠 Tecnologias
- **Linguagem:** Python
- **Framework:** Streamlit
- **Processamento:** Pandas
- **Monitoramento:** UptimeRobot (Garantia de uptime 24/7)

## 📋 Como funciona
O app lê uma planilha de dados (formato .csv) e transforma linhas complexas em cards visuais. O processamento ocorre via Streamlit Cloud, garantindo acesso via navegador em qualquer dispositivo móvel.
